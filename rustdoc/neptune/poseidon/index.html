Poseidon hash

Structs§

  • Poseidon
    Holds preimage, some utility offsets and counters along with the reference to PoseidonConstants required for hashing. Poseidon is parameterized by ff::PrimeField and Arity, which should be similar to PoseidonConstants.
  • PoseidonConstants
    Holds constant values required for further Poseidon hashing. It contains MDS matrices, round constants and numbers, parameters that specify security level (Strength) and domain separation (HashType). Additional constants related to optimizations are also included.
