This module contains the logic for fetching the proofs parameters from the network. As a general rule, the parameters are first fetched from ChainSafeâ€™s Cloudflare R2 bucket, if that fails (or is overridden by PROOFS_ONLY_IPFS_GATEWAY_ENV), the IPFS gateway is used as a fallback.

The reason for this is that the IPFS gateway is not as reliable and performant as the centralized solution, which contributed to issues in CI in the past.
