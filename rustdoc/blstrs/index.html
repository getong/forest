§blstrs

An implementation of the BLS12-381 pairing-friendly elliptic curve construction.

Structs§

  • Bls12
    Bls12-381 engine
  • Fp
    Fp values are always in Montgomery form; i.e., Fp(a) = aR mod p, with R = 2^384. blst_fp.l is in little-endian u64 limbs format.
  • Fp2
  • Fp12
    This represents an element $c_0 + c_1 w$ of $\mathbb{F}{p^12} = \mathbb{F}{p^6} / w^2 - v$.
