Struct libp2p_dns::Transport

source · 
pub struct Transport<T, R> { /* private fields */ }
Expand description

A Transport for performing DNS lookups when dialing Multiaddresses. You shouldn’t need to use this type directly. Use tokio::Transport or [async_std::Transport] instead.
